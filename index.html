CSS Explained

Introduction

CSS (Cascading Style Sheets) is a stylesheet language used for describing the look and formatting of a document written in HTML. It allows you to separate the presentation and layout of a web page from its structure.

Benefits of CSS

Here are some benefits of using CSS:

  • Consistency: CSS enables consistent styling across multiple web pages.
  • Efficiency: CSS allows you to apply styles to multiple elements at once.
  • Maintainability: By separating style from structure, it's easier to update and maintain a website.

How to Use CSS

To use CSS, you need to define styles and apply them to HTML elements. Here's an example:

      
        <!-- HTML code -->
        <h1>Hello, World!</h1>
        
        /* CSS code */
        h1 {
          color: red;
          font-size: 24px;
        }

In the example above, the CSS code selects the <h1> element and applies styles to it. The color property sets the text color to red, and the font-size property sets the font size to 24 pixels.

Selectors

CSS uses selectors to target HTML elements. Here are some commonly used selectors:

  • Element selector: h1 selects all <h1> elements.
  • Class selector: .container selects all elements with the class "container".
  • ID selector: #header selects the element with the ID "header".
